Features

Everything you need. Nothing you don't.

Built with a single principle: your data stays on your device, always.

🔒

Privacy First

Uses activeTab only — zero access to any page until you click the extension icon. No background access, no tracking, no telemetry. 100% ad-free.

🎯

Multiple Clickers

Add up to 50 parallel clickers per website, each with its own label, interval, and target element. Perfect for complex idle games.

⚡

Ultra-Fast Clicking

Minimum interval of 25ms — that's 40 clicks per second. Automatically bypasses anti-cheat cooldown variables like Cookie Clicker's Game.lastClick.

👁

Live Highlighting

Hover over any clicker card in the popup and the corresponding element on the page instantly glows pink — so you always know exactly what's being clicked.

💾

State Resumption

Clicker configurations persist per-site via chrome.storage.local. After a page reload, just open the popup — all your clickers resume automatically.

🌐

Cross-Browser

Works seamlessly on Chrome, Firefox, Edge, Brave, and any Chromium-based browser. Manifest V3 compliant. GitHub Actions builds both Chrome and Firefox ZIPs automatically.

How it works

Up and running in 3 steps.

No configuration files, no accounts, no cloud. Just install and click.

01

Install the Extension

Download directly from the Chrome Web Store, or grab the latest .zip from GitHub Releases. For Chrome/Edge/Brave, enable Developer mode and hit Load unpacked. For Firefox, load via about:debugging.

02

Select an Element

Navigate to your game. Click the KoalaClicker icon in the toolbar, then press "Add New Clicker". The popup closes and Element Selection Mode activates — a banner appears on the page.

Hover over any element to see it glow pink, then click to register it as a clicker target.

03

Click Automatically

The popup shows all your active clickers. Adjust the interval, rename the clicker, toggle individual ones on or off — all in real time. Hover a card and the corresponding element lights up on the page.

All settings save automatically. After a page reload, just open the popup and everything resumes.
