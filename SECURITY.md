# Security Policy

## Supported Versions

At this time, only the latest release of the KoalaClicker extension is supported with security updates. We highly recommend always using the latest version available on the Releases page.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Security and privacy are the core foundations of KoalaClicker. If you discover a security vulnerability or a privacy leak within this project, please report it immediately.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please send an email or directly contact the repository maintainer (`Shik3i`). You should receive a response within 48 hours. If the issue is confirmed, a patch will be developed and released as quickly as possible.

### What to include in your report:
- A detailed description of the vulnerability or privacy issue.
- The steps required to reproduce the issue.
- The browser and operating system version you are using.
- Any potential impact on users.

Thank you for helping keep KoalaClicker safe and secure!
