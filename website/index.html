Features Funktionen

Everything you need. Nothing you don't. Alles was du brauchst. Sonst nichts.

Built with a single principle: your data stays on your device, always. Entwickelt nach einem klaren Prinzip: Deine Daten gehören dir und bleiben lokal.

🔒

Privacy First Privatsphäre zuerst

Uses activeTab only — zero access to any page until you click the extension icon. No background access, no tracking, no telemetry. 100% ad-free. Nutzt nur activeTab — kein Zugriff auf Webseiten, bis du auf das Symbol klickst. Keine Hintergrund-Rechte, kein Tracking, 100% werbefrei.

🎯

Multiple Clickers Mehrere Clicker

Add up to 50 parallel clickers per website, each with its own label, interval, and target element. Perfect for complex idle games. Füge bis zu 50 parallele Clicker pro Webseite hinzu, jeder mit eigenem Namen, Intervall und Element. Perfekt für komplexe Idle-Games.

⚡

Ultra-Fast Clicking Extrem schnelles Klicken

Minimum interval of 25ms — that's 40 clicks per second. Automatically bypasses anti-cheat cooldown variables like Cookie Clicker's Game.lastClick. Mindestintervall von 25 ms — das sind 40 Klicks pro Sekunde. Umgeht automatisch Anti-Cheat-Sperren wie Game.lastClick in Cookie Clicker.

👁

Live Highlighting Live-Hervorhebung

Hover over any clicker card in the popup and the corresponding element on the page instantly glows pink — so you always know exactly what's being clicked. Bewege die Maus über eine Clicker-Karte und das Element auf der Seite leuchtet pink auf — so weißt du immer genau, was angeklickt wird.

💾

State Resumption Speichern & Laden

Clicker configurations persist per-site via chrome.storage.local. After a page reload, just open the popup — all your clickers resume automatically. Deine Clicker werden pro Webseite über chrome.storage.local gespeichert. Nach einem Neuladen sind alle deine Clicker sofort wieder da.

🌐

Cross-Browser Browser-übergreifend

Works seamlessly on Chrome, Firefox, Edge, Brave, and any Chromium-based browser. Manifest V3 compliant. GitHub Actions builds both Chrome and Firefox ZIPs automatically. Läuft problemlos auf Chrome, Firefox, Edge, Brave und allen Chromium-Browsern. Manifest V3 konform. GitHub Actions baut Chrome- und Firefox-ZIPs vollautomatisch.

How it works So funktioniert's

Up and running in 3 steps. In 3 Schritten startklar.

No configuration files, no accounts, no cloud. Just install and click. Keine Konfigurationsdateien, keine Accounts, keine Cloud. Einfach installieren und loslegen.

01

Install the Extension Erweiterung installieren

Download directly from the Chrome Web Store, or grab the latest .zip from GitHub Releases. For Chrome/Edge/Brave, enable Developer mode and hit Load unpacked. For Firefox, load via about:debugging. Lade sie aus dem Chrome Web Store oder als .zip aus den GitHub-Releases. Aktiviere für Chrome/Edge/Brave den Entwicklermodus und klicke auf "Entpackte Erweiterung laden". Für Firefox nutze about:debugging.

02

Select an Element Element auswählen

Navigate to your game. Click the KoalaClicker icon in the toolbar, then press "Add New Clicker". The popup closes and Element Selection Mode activates — a banner appears on the page. Gehe auf deine Spiel-Seite. Klicke auf das KoalaClicker-Symbol in der Symbolleiste, dann auf "Add New Clicker". Das Popup schließt sich und der Auswahlmodus startet — ein Banner erscheint.

Hover over any element to see it glow pink, then click to register it as a clicker target. Bewege die Maus über ein Element, bis es pink leuchtet, und klicke darauf, um es als Klick-Ziel zu speichern.

03

Click Automatically Automatisch klicken

The popup shows all your active clickers. Adjust the interval, rename the clicker, toggle individual ones on or off — all in real time. Hover a card and the corresponding element lights up on the page. Das Popup zeigt dir alle aktiven Clicker. Passe das Intervall an, benenne sie um, schalte sie an oder aus — alles in Echtzeit. Fahre über eine Karte, um das Element auf der Seite zu sehen.

All settings save automatically. After a page reload, just open the popup and everything resumes. Alle Einstellungen werden automatisch gespeichert. Nach einem Neuladen der Seite einfach das Popup öffnen und alles läuft weiter.
