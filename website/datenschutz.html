🇩🇪 Datenschutzerklärung

1. Verantwortlicher

KoalaClicker ist ein privates Open-Source-Hobbyprojekt. Die Browserextension wird ohne kommerzielle Absicht entwickelt und kostenlos zur Verfügung gestellt.

2. Grundsatz: Keine Datenerhebung

Wir erheben, speichern oder übermitteln keinerlei personenbezogene Daten über die Extension. KoalaClicker arbeitet vollständig lokal auf Ihrem Gerät. Es gibt keine externen Server für die Extension, keine Analytics, keine Telemetrie, keine Werbung und kein Tracking – in keiner Form.

Die Extension macht keinerlei Netzwerkanfragen nach außen. Es werden keine Daten Ihres Geräts oder Ihres Browsings an Dritte übermittelt.

3. Web-Hosting & Server-Logs

Diese Informations-Webseite wird auf einem Server der Hetzner Online GmbH in deren Rechenzentrum in Helsinki, Finnland gehostet.

Beim Zugriff auf diese Webseite erfasst der Webserver automatisch typische Verbindungsdaten (Server-Logfiles). Dazu gehören:

  • Ihre IP-Adresse
  • Datum und Uhrzeit des Zugriffs
  • Aufgerufene Seite/Datei
  • HTTP-Statuscode
  • Browsertyp und Browserversion (User-Agent)
  • Referrer-URL (die zuvor besuchte Seite, sofern übermittelt)

Diese Daten werden ausschließlich zur Aufrechterhaltung des technischen Betriebs, zur Systemsicherheit und zur Fehleranalyse erhoben. Die Server-Logfiles werden automatisch nach 7 Tagen rotiert bzw. vollständig gelöscht.

Es findet keine Zusammenführung dieser Daten mit anderen Datenquellen statt. Diese Logs werden zu keinem Zeitpunkt personalisiert, analysiert oder mit einer Person verknüpft oder an Dritte weitergegeben.

4. Lokaler Speicher (chrome.storage.local)

KoalaClicker verwendet chrome.storage.local ausschließlich, um Ihre konfigurierten Clicker (CSS-Selektoren, Intervalle und Namen) lokal auf Ihrem Gerät zu speichern. Diese Daten:

  • verlassen Ihr Gerät niemals
  • werden nicht an Server übertragen
  • werden nicht mit Dritten geteilt
  • können jederzeit durch Löschen der Extension vollständig entfernt werden

5. Berechtigungen

KoalaClicker benötigt folgende Browser-Berechtigungen, die ausschließlich für die beschriebenen Funktionen genutzt werden:

  • activeTab — Erlaubt den Zugriff auf die aktuelle Webseite ausschließlich dann, wenn Sie explizit auf das Extension-Symbol klicken. Die Extension hat keinen Hintergrundzugriff, kann Ihren Browserverlauf nicht lesen und läuft nicht automatisch auf Seiten.
  • storage — Wird genutzt, um Ihre Clicker-Konfigurationen lokal auf Ihrem Gerät zu speichern, sodass diese beim erneuten Öffnen der Extension wiederhergestellt werden können.
  • scripting — Wird genutzt, um die Auto-Klick-Logik in den aktiven Tab zu injizieren, den Sie ausgewählt haben.

6. Drittanbieter

KoalaClicker integriert keinerlei Drittanbieter-Dienste, APIs, Analysetools, externe Schriftarten oder Tracking-Pixel. Die Extension ist vollständig ohne externe Abhängigkeiten gebaut.

7. Open Source

Der gesamte Quellcode ist öffentlich auf GitHub einsehbar und auditierbar. Jeder kann die Funktionsweise der Extension unabhängig prüfen.

8. Änderungen dieser Datenschutzerklärung

Sollten sich Änderungen ergeben, wird das Datum am Anfang dieses Dokuments aktualisiert. Da wir keine Kontaktdaten erheben, empfehlen wir, diese Seite gelegentlich zu prüfen.

9. Kontakt

Bei Fragen zum Datenschutz öffnen Sie bitte ein Issue auf unserem GitHub-Repository oder kontaktieren Sie uns per E-Mail.

🇬🇧 Privacy Policy

1. Data Controller

KoalaClicker is a private, open-source hobby project. The browser extension is developed and provided free of charge with no commercial intent.

2. Core Principle: We Collect No Data

We do not collect, store, or transmit any personal data whatsoever via the extension. KoalaClicker operates entirely locally on your device. There are no external servers for the extension, no analytics, no telemetry, no advertising, and no tracking of any kind.

The extension makes zero outbound network requests. No data about your device or your browsing is ever sent to any third party.

3. Web Hosting & Server Logs

This informational landing page website is hosted on a server provided by Hetzner Online GmbH in their datacenter in Helsinki, Finland.

When you access this website, the web server automatically records typical connection data (server log files). This includes:

  • Your IP address
  • Date and time of access
  • Page/file accessed
  • HTTP status code
  • Browser type and version (user agent)
  • Referrer URL (the previously visited page, if transmitted)

This data is collected solely for maintaining technical operations, system security, and troubleshooting. The server log files are automatically rotated and completely deleted after 7 days.

We do not combine this data with other data sources. These logs are never personalized, analyzed, linked to any individual, or shared with third parties.

4. Local Storage (chrome.storage.local)

KoalaClicker uses chrome.storage.local exclusively to save your configured clickers (CSS selectors, intervals, and names) locally on your device. This data:

  • Never leaves your device
  • Is never transmitted to any server
  • Is never shared with any third party
  • Can be fully removed at any time by uninstalling the extension

5. Permissions Explained

KoalaClicker requires the following browser permissions, used solely for their described purpose:

  • activeTab — Grants access to the current webpage only when you explicitly click the extension icon. The extension has no background access, cannot read your browsing history, and does not automatically run on any page.
  • storage — Used to save your clicker configurations locally on your device, so they can be restored when you reopen the extension.
  • scripting — Used to inject the auto-clicking logic into the active tab you have chosen to interact with.

6. Third-Party Services

KoalaClicker does not integrate with any third-party services, APIs, analytics tools, external fonts, or tracking pixels. The extension is built entirely with zero external dependencies.

7. Open Source

The entire source code is publicly available and auditable on GitHub. Anyone can independently verify exactly how the extension works.

8. Changes to This Policy

If this Privacy Policy is updated, the date at the top of this document will be revised. Since we do not collect contact information, we encourage you to review this page periodically.

9. Contact

If you have any questions about this Privacy Policy, please open an issue on our GitHub repository or contact us via email.
